Descripción

En este curso vamos a aprender a implementar de forma sencilla y rapida las librerías mas populares y utilizadas por desarrolladores frontend tales como Google Maps, Spotify SDK, Login con redes sociales (Facebook y Google), utilizandolas en los 4 proyectos incluidos.

En cada uno de los proyectos utilizaremos los conceptos esenciales de React Js acompañados de librerias de gran potencial tales como Redux para el control de los estados de nuestra app, Axios para la creacion de peticiones http y React Router para el control de las distintas pantallas de nuestros proyectos. Ademas de Materialize CSS para los estilos.
